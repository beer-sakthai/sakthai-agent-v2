# B2B Deal Packaging and Pro Pricing

## 1. Lead with the value metric

Pick **one** primary value metric the customer can see growing with their
success:

| Value metric | Best for | Watch-outs |
|---|---|---|
| Seats | HR, security, sales tools | Under-monetizes power users |
| Usage/events | API, observability, automation | Volatility / surprise bills |
| Workloads/projects | Dev tools, analytics | Must map to budget owner |
| Revenue managed | Fintech, billing, commerce | Customer audits, complex |

The package is then built around **allowances** of that metric.

## 2. Good-Better-Best + Enterprise

Use 3-4 tiers. Keep them simple enough for a sales rep to explain in 30
seconds:

| Tier | Goal | Typical composition |
|---|---|---|
| **Starter/Pro** | Land, self-serve | Core features, limited seats/usage |
| **Business/Growth** | Team-wide adoption | Admin, SSO, reporting, higher caps |
| **Enterprise** | Custom terms | Security, SLAs, dedicated support, negotiated price |

Rules:
- The jump from Business to Enterprise should be **>30%** to separate tiers.
- Each tier must have **at least one gate** the buyer clearly wants.
- Never let Enterprise be the only option for a standard security feature.

## 3. Build the proposal: 4-block structure

1. **Business case** — quantified ROI for the customer.
2. **Package map** — what is included and what is not.
3. **Price & terms** — list, discount, net, payment terms.
4. **Governance** — approval path, escalation, renewal terms.

## 4. Discount guardrails

Set hard rules before any deal desk review:

- **Maximum standard discount** (e.g., 20%)
- **Non-standard discount** requires CFO + VP Sales sign-off
- **No discount without a trade**: longer term, case study, prepay, logo rights, expansion commitment
- **Publish annual price uplift** (e.g., 5-8%) in all multi-year deals

### Discount trade menu

| Trade | Value |
|---|---|
| 2-year term | Lock in price, improve retention |
| Pre-pay annually | Cash flow, lower churn |
| Case study / logo | Marketing value |
| Expansion commitment | Predictable NRR |
| Reference calls | Sales enablement |

## 5. Renewal and expansion ramps

- Front-load expansion **at renewal**, not mid-term, to preserve value.
- Use **step-up pricing**: Year 1 discounted, Year 2 list, Year 3 list + uplift.
- Include **true-up language** for overage at 1.2× list price.

## 6. Red flags

- Discount > 40% without multi-year/prepay → likely bad unit economics.
- Deal where Professional Services > 30% of ACV → scope creep risk.
- Custom terms that require engineering → model as separate SOW.
- Renewal discount > new-logo discount → encourages churn-and-return.

## 7. Verification checklist

Before sending a proposal:

- [ ] Value metric is measurable and customer-visible.
- [ ] Package boundaries are clear (what is in/out).
- [ ] List price is defensible against alternatives.
- [ ] Net price hits gross-margin target including CS costs.
- [ ] Contract includes annual uplift or expansion language.
- [ ] Deal desk/CFO approval documented for non-standard terms.
