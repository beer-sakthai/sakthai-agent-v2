# sakthai-coding-skill-authoring

A skill is a SKILL.md with valid YAML frontmatter (name, description, version) and a focused body. Keep `metadata.sakthai.tags` accurate, validate with `sakthai skills validate`, and describe *when* to use the skill, not just what it is.
